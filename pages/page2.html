Le Css c'est quoi ?

  Si le Html peut se définir comme étant la structure de base d'une page web ou son squelette, le Css, abréviation de Cascading Style Sheets (feuilles de style en cascade), se définit plutôt comme étant la peau et va permettre ainsi de modifier l'allure d'une page en changeant la couleur du texte, inserer des images, modifier l'agencement des éléments entre

Syntaxe de CSS

 CSS est un langage basé sur des règles . On définit des règles de styles destinées à des éléments ou des groupes d'éléments particuliers dans la page. Par exemple, « Je veux que les paragraphes de ma page s'affichent en rouge. »

  La règle commence par choisir un sélecteur, pour "sélectionner" l'élément HTML mis en forme. Ici, le sélecteur est p (paragraphe), il s'agit d'un sélecteur de type (notons qu'il existe plusieurs types de sélecteurs mais nous n'aborderons pas le sujet pour l'instant).

  le style s'applique à tous les p de la page. Suit une paire d'accolades { } à l'intérieur desquelles on trouve une ou plusieurs déclarations, sous la forme d'une paire propriété: valeur (dans notre exemple la propriété est color et la valeur est red). Nous voyons bien dans l'image présentée que le texte "Mon cours de css numéro" s'affiche en rouge.
