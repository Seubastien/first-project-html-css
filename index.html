Objectjif du site

L'objectif de ce site est de rendre les concepts du développement web accessibles aux débutants. Le but est également de comprendre la diférence entre les différentes technologies (PHP, JavaScript, HTML, CSS). Chacune d'entre elles sera expliquée de manière simple et accompagnée d'exemples concrets pour illustrer son utilisation.
